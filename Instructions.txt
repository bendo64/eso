--Server--

*Connect to network ID : 65228d8d6d2afb9d

*Contact me to be authenticated on the vpn network

--Game--
*For Chrome OS:
	*Activate linux environment in settings
	*Install git by executing "sudo apt install git" in terminal
	*Download "eso_setup.sh" script from "github.com/bendo64/eso/tree/main/linux_scripts" and paste it into your "Linux files" folder in the files app
	*Execute the following command in the linux terminal:
		sudo chmod +x eso_setup.sh
	*Execute "sudo ./eso_setup.sh" to install eso
	
	
*For Windows:
	*Grab .exe file from "github.com/bendo64/endostellar_online/tree/main/Windows releases"
	
	*Alternatively install git on windows, open up a git terminal
	*Enter "git clone https://github.com/bendo64/eso.git"
	*Locate "eso" folder to access latest exe version in Windows Releases folder, and use the update.bat script to auto update
